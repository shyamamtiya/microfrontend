# microfrontend
Microfrontend using React

Container App (using React.js)
  Steps to run main container : <br/>
  cd container <br/>
  npm i  <br/>
  npm start <br/>
  http://localhost:8080/ <br/>
  
 Dashoard App(using React.js) <br/>
   Steps to run Dashboard app <br/>
    cd dashboard <br/>
    npm i  <br/>
    npm start <br/>
    http://localhost:8082/ <br/>
